🌾 Agriculture Data Analytics Project

(Snowflake + AWS + Power BI Dashboard)

📌 Project Overview

This project analyzes agricultural data including Rainfall, Temperature, Humidity, and Crop Yield across different:

📅 Years

🌦 Seasons

🌱 Crops

📍 Locations

The goal of this project is to build a cloud-based data analytics pipeline and create an interactive dashboard to identify trends and insights affecting crop yield.

🛠 Tools & Technologies Used

☁ Cloud Platform: Amazon Web Services (AWS)

🗄 Data Warehouse: Snowflake

📊 Data Visualization: Microsoft Power BI

🐍 SQL for data transformation

📁 CSV Dataset

🏗 Architecture Flow

Raw CSV data uploaded to AWS (S3)

Data loaded into Snowflake

Data cleaning and transformation using SQL

Aggregations created:

Average Rainfall by Year

Average Temperature by Season

Average Humidity by Location

Average Crop Yield by Crop Type

Connected Snowflake to Power BI

Built interactive dashboard with filters

📊 Dashboard Features

📈 Year-wise average rainfall trends

🌡 Temperature comparison by season

💧 Humidity analysis by location

🌾 Crop yield performance analysis

🎛 Interactive filters (Year, Season, Crop, Location)

🔍 Key Insights

Identified seasonal patterns affecting crop yield

Observed correlation between rainfall and productivity

Compared regional agricultural performance

Highlighted high-yield crops under specific climate conditions

🚀 How to Run This Project

Upload CSV file to AWS S3

Create Snowflake table and load data

Run transformation queries

Connect Power BI to Snowflake

Import dataset and build visuals
